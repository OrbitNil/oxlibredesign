## 🚀 Getting Started

### npm Package

```bash
npm install @communityox/ox_lib
```

## 🖥️ Development Setup

Enhance your development experience with these recommended tools:

1. **[Lua Language Server](https://marketplace.visualstudio.com/items?itemName=sumneko.lua)** — Annotations, type checking, diagnostics, and more.
2. **[CfxLua IntelliSense](https://marketplace.visualstudio.com/items?itemName=communityox.cfxlua-vscode-cox)** — Adds natives and cfxlua runtime declarations to LLS.

To load ox_lib into your global development environment, add the resource path to your workspace or user settings:

```json
"Lua.workspace.library": ["c:/fxserver/resources/ox_lib"]
```
